Siege Game Instructions

1) Unzip the archive
2) Using the command line, go to the /src/com/eecs285/siegegame folder
3) Compile the server by entering ‘javac Server.java MiniServer.java’
4) Using the command line, go to the /src folder
5) Start the server by typing ‘java com.eecs285.siegegame.Server x’, where x is an int
   that represents the number of players that will be partaking (usually 2-4).
6) Open Eclipse and use /src as the working directory (each client must do the next steps)
7) Open Siege.java and change the final IPaddress string at the top of the file to the IP     
   address of the server
8) Save and execute Siege.java
9) Follow game directions
